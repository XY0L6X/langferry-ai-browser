# 文渡 使用指南

文渡 是一款 AI 翻译浏览器——在手机上看外文网页时，一键把整个页面翻译成中文。

---

## 1. 安装应用

### 方式一：直接安装 APK（推荐）
将 `app-debug.apk` 传到手机上，点击安装。如提示"未知来源"，在设置中允许"安装未知应用"即可。

### 方式二：从源码构建
参考文末「开发者附录」。

---

## 2. 配置 API 密钥（必须操作）

> ⚠️ **这是最重要的一步！** 翻译功能需要调用 AI 大模型，你必须先获取一个 API 密钥（也叫 API Key），类似一把"钥匙"让应用能使用 AI 服务。

### 2.1 什么是 API Key？
可以理解为 AI 服务的"密码"。你在 AI 平台注册后获得一串字符，填入应用就能使用翻译功能。**Key 的花费很低**——翻译几十个网页通常只需几分钱。

### 2.2 获取 API Key（推荐 DeepSeek，国内友好，价格极低）

**推荐方案：DeepSeek**（国内可直接访问，注册送免费额度）

1. 打开浏览器访问：**https://platform.deepseek.com/**
2. 点击右上角「注册」，用手机号或邮箱注册账号
3. 注册完成后登录，进入控制台
4. 点击左侧菜单「API Keys」
5. 点击「创建 API Key」，输入名称（如"文渡"），点击确定
6. **复制生成的 Key**（以 `sk-` 开头的一长串英文字符），妥善保存

> 💰 **费用说明**：DeepSeek V4 Flash 模型翻译一个网页约花费 0.001-0.01 元。新用户通常赠送 10 元额度，足够翻译上千个网页。

**备选方案：通义千问（阿里云）**

1. 访问：**https://dashscope.console.aliyun.com/**
2. 用阿里云账号登录（没有就先注册）
3. 在左侧菜单找到「API-KEY 管理」
4. 创建并复制 API Key

> 其他支持的平台：OpenAI、Mimo AI 等，操作流程类似。只要提供 OpenAI 兼容接口的 API 都可以使用。

### 2.3 在应用中填入 API Key

1. 打开 文渡 应用，点击底部导航栏右侧的「**设置**」图标
2. 在设置页面点击「**API 配置**」
3. 点击「**添加 API 配置**」（或右上角 + 号）
4. 在选择预设模板下拉框中，选择你注册的平台（如「DeepSeek V4 Flash」）
   - 名称、API 端点、模型名称会自动填入
5. 在「**API Key**」输入框中粘贴你刚才复制的 Key（以 `sk-` 开头的那串字符）
6. 点击「**测试连接**」按钮验证 Key 是否有效
   - 看到 ✅ "连接成功" 说明配置正确
   - 看到 ❌ 失败提示，检查 Key 是否复制完整、网络是否正常
7. 测试成功后，点击右上角「**保存**」
8. 返回设置页，确保刚保存的配置旁有 ✓ 标记表示已激活

> 💡 **提示**：API Key 就像银行卡密码，不要随意分享给他人。

## 3. 使用浏览器

### 3.1 打开网页
- 在顶部地址栏输入网址（如 `www.ted.com`）
- 也可以输入中文关键词直接搜索
- 底部导航栏有前进、后退、刷新按钮

### 3.2 多标签页
- 点击地址栏右侧的数字按钮可管理多个标签页
- 支持新建、切换、关闭标签页

---

## 4. 翻译网页

### 4.1 全页翻译
1. 在浏览器中打开一个外文网页
2. 点击底部导航栏中间的「**翻译**」按钮（带翻译图标）
3. 等待翻译完成（页面会逐块变为中文）
4. 翻译完成后按钮显示"翻译完成"

### 4.2 切换显示模式
翻译完成后，再次点击翻译按钮会弹出选项：

| 选项 | 效果 |
|------|------|
| 仅译文 | 只显示中文翻译 |
| 原文译文对照 | 同时显示原文和译文 |
| 显示原文 | 恢复原始外语页面 |
| 重新翻译 | 重新提取并翻译页面（适用于点开"展开"加载更多内容后） |

### 4.3 划词翻译
- 长按网页中的外文文字
- 选中要翻译的文本
- 底部自动弹出翻译结果
- 支持复制译文或收藏

---

## 5. 下载管理

- 浏览网页时点击可下载的链接（PDF、图片等），会自动开始下载
- 在底部导航栏「下载」tab 可查看下载列表
- 点击下载项右侧菜单可「打开文件」或「删除」
- 删除时可选「仅删记录」或「同时删除文件」
- 点击「更改」可自定义下载目录

## 6. 常见问题

### Q：翻译按钮点了没反应？
1. 确认已配置 API Key（设置 → API 配置 → 添加 → 测试连接成功 → 保存）
2. 确认当前有网络连接
3. 确认页面已加载完成（地址栏不再显示加载条）

### Q：翻译结果显示"翻译失败"？
1. 检查 API Key 是否过期或用完额度
2. 进入设置 → API 配置，点击「测试连接」验证
3. 如果测试也失败，可能是 Key 失效，需重新获取

### Q：翻译速度慢？
- 已优化为并发翻译，常规页面约需 5-30 秒
- 页面越复杂、文本越多，耗时越长
- 第二次翻译同一页面会更快（有缓存）

### Q：翻译后页面某些部分没翻译？
- 部分网页的动态内容（如评论区）可能不会被提取
- 点击翻译按钮 → 选择「重新翻译」可覆盖新加载的内容
- 如果"展开"按钮加载了新内容，需重新翻译

### Q：需要一直开着网络吗？
- 浏览网页和翻译功能都需要网络连接
- 翻译结果会缓存，同一段文字第二次翻译不需要联网

### Q：API Key 安全吗？
- Key 保存在手机本地，不会上传到任何第三方服务器
- 不要将 Key 分享给他人

### Q：费用会不会很高？
- 翻译是"按量计费"，只在实际翻译时才扣费
- 常用模型（如 DeepSeek V4 Flash）价格极低，正常使用每月不到 1 元

### Q：能否翻译成其他语言？
- 当前默认翻译为中文
- 后续版本将支持选择目标语言

---

## 7. 收藏与历史

- **收藏**：点击地址栏右侧的收藏图标，可将当前网页保存到收藏夹
- **历史**：每次浏览和翻译的网页自动记录在历史 tab 中
- 支持搜索、分类管理和批量删除

## 开发者附录

### 构建 APK

```bash
# 配置 Flutter 环境（需 Flutter 3.22+）
cd langferry
flutter pub get

# 构建调试版 APK
flutter build apk --debug

# APK 生成路径：build/app/outputs/flutter-apk/app-debug.apk
```

### 项目结构

```
langferry/
├── assets/js/              # JavaScript 注入脚本
│   ├── extract_text.js     # 网页文本提取
│   └── replace_text.js     # 文本替换与模式切换
├── lib/
│   ├── main.dart           # 入口
│   ├── app.dart            # MaterialApp 配置
│   ├── core/services/      # 核心服务（API/下载/翻译/JS注入）
│   ├── providers/          # 状态管理（Riverpod）
│   ├── pages/              # 页面（浏览器/历史/收藏/下载/设置）
│   └── widgets/            # 通用组件
└── pubspec.yaml            # 依赖配置
```
